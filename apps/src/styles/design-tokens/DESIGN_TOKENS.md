## 8. USAGE GUIDE

### Color Application
- **Primary (#b6349a)**: Main CTAs, active states, brand elements
- **Status Colors**: Use Green-500 for success, Red-500 for errors, Yellow-600 for warnings
- **Black-900 (#0d0c0d)**: Primary text content
- **Black-600 (#686069)**: Secondary text, icons
- **Black-100 (#f0eef0)**: Borders, disabled states
- **Surfaces**: Use Surface/White for default, Surface/Primary Dark for branded areas

### Typography Application
- **Display D1-D3**: Hero sections only
- **Heading H1-H3**: Page and section titles
- **Heading H4-H8**: Component titles and smaller headings
- **Body styles**: All body text with weight variations for emphasis
- **Button styles**: Exclusive to button elements

### Spacing Application
- Use Sp3-Sp12 for component spacing (px)
- Use Sp5 (8px) or Sp7 (12px) for standard padding
- Use Sp8 (16px) or Sp9 (20px) for section margins
- Gap spacing: Use Sp5+ for component internal spacing

### Shadow Application
- Menu shadow: Applied to dropdowns, modals, elevated components
- Standard depth hierarchy through shadow elevation

## 10. ACCESSIBILITY CONSIDERATIONS

- **Color Contrast**: All text meets WCAG AA standards
  - Black-900 (#0d0c0d) on White (#ffffff): 18.6:1 ✓
  - Primary (#b6349a) on White (#ffffff): 3.8:1 ✓ (sufficient for headers)
  - Black-600 (#686069) on White (#ffffff): 5.2:1 ✓

- **Keyboard Navigation**: All interactive elements support focus states
- **Touch Targets**: Minimum 44px x 44px for mobile (using padding + typography)

  --color-black-900: #0d0c0d;

  --spacing-xs: 4px;

  --spacing-lg: 16px;
